# Lwjgl Fusion
Compatibility layer between lwjgl2 and lwjgl3

## Differences from Legacy-Lwjgl3
* Latest Lwjgl3
* Delete Fabric Mod
* Wayland support (not complete)

## Why should use this?
* Improved performance is expected
* Can use the latest lwjgl3 features (NanoVG, nfd etc...)
